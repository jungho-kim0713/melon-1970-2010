# 통·統·通 교사연구회 프로젝트 문서

## 1. 프로젝트 목적
**"대중가요 가사 빅데이터 분석을 통한 세대 공감 통계 수업 모델 개발"**
- 1970년대부터 2020년대까지의 인기 대중가요 가사를 데이터로 변환하여 세대별 문화적, 사회적 가치관의 차이를 객관적으로 분석합니다.
- 학생들은 생성형 AI(NotebookLM)를 활용하여 텍스트 마이닝을 수행하고, 공학도구(통그라미)로 데이터를 시각화함으로써 비판적 사고력과 데이터 문해력을 기릅니다.
- 최종적으로 통계 데이터를 근거로 세대 간의 차이를 이해하고 공감하는 상호 존중의 태도를 함양하는 융합 수업(정보, 국어, 일반사회) 모델을 제시하는 것이 목적입니다.

## 3. 핵심 워크플로우

본 프로젝트의 수업 및 연구 모델은 다음 4단계 파이프라인으로 구성됩니다.

### Step 1. 데이터 수집 (Data Crawling)
- `melon_crawler.py`를 실행하여 1970년대부터 2020년대까지 각 연대별 상위 100곡의 메타 정보(곡명, 가수명)와 **전체 가사 텍스트**를 자동 수집합니다.
- 수집된 데이터는 CSV 및 JSON 형태로 정제되어 저장됩니다.

### Step 2. AI 텍스트 마이닝 (Data Processing with NotebookLM)
- 수집한 가사 데이터(CSV/JSON)를 구글 NotebookLM에 업로드합니다.
- AI 프롬프트를 활용하여 연대별 핵심 키워드 빈도 추출, 긍정/부정 감정 지수 분석, 사회적 가치관 도출 등 비정형 텍스트 데이터를 정형 데이터로 변환합니다.

### Step 3. 통계 시각화 (Data Visualization with 통그라미)
- NotebookLM을 통해 추출된 데이터(단어 빈도, 감정 지수 등)를 통계청 공학도구인 **통그라미**에 입력합니다.
- 세대별 특징을 직관적으로 비교할 수 있는 다각적인 그래프(감정 온도계, 시대별 변화 추이선 등)를 제작합니다.

### Step 4. 융합 수업 및 컨퍼런스 (Class & Reflection)
- 학생들이 통계적 시각화 자료를 바탕으로 다른 세대의 시대적 맥락과 가치관을 해석합니다.
- '세대 공감 통계 포스터'를 제작하고 발표하여, 세대 갈등 완화를 위한 상호 이해와 윤리적 토론을 진행합니다.

## 4. 파이썬 스크립트 실행 방법

멜론 차트 크롤링 스크립트를 실행하여 최신 가사 데이터를 수집하는 방법입니다.

### 4.1. 필수 라이브러리 설치
크롤링에 필요한 HTML 파싱 라이브러리를 설치해야 합니다. VSCode 터미널 또는 명령 프롬프트(CMD)를 열고 다음 명령어를 입력합니다.
```bash
pip install beautifulsoup4
```

### 4.2. 크롤러 실행
스크립트가 위치한 폴더로 이동한 뒤 파이썬 파일을 실행합니다.
```bash
cd "f:\Kim_Jungho\jungho_webapp\통.통.통 교사연구회"
python melon_crawler.py
```

### 4.3. 실행 시 주의사항
- 멜론 서버의 봇(Bot) 차단을 우회하기 위해 `User-Agent` 헤더를 포함하고 있으며, 한 곡을 수집할 때마다 **1초의 지연 시간(Delay)**이 설정되어 있습니다.
- 총 600곡(6개 연대 × 100곡)을 수집하므로 **약 10분에서 15분 정도** 소요됩니다. 스크립트가 돌아가는 동안 터미널 창을 닫지 말고 대기해 주세요.
- 실행이 모두 완료되면 현재 폴더에 `melon_all_decades_lyrics.csv` 파일이 생성됩니다. 이 파일을 NotebookLM에 업로드하여 분석을 진행하시면 됩니다.
